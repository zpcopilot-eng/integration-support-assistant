# QR Code Payment Documentation

Nguồn: https://docs.zalopay.vn/docs/guides/payment-acceptance/qrcode/intro

## Overview

Zalopay's QR Code Payment feature enables secure smartphone-based transactions. Users can scan QR codes displayed at point-of-sale terminals, websites, or invoices using the Zalopay app to complete purchases. According to the documentation, this method is described as "fast, easy, and reduces the need for physical contact or cash."

## Integration Options

Zalopay provides two distinct integration approaches:

1. **Static QR Code** — A no-code solution for merchants seeking straightforward payment acceptance without technical implementation requirements.

2. **Dynamic QR Code** — A more sophisticated integration option offering expanded functionality, though requiring greater technical complexity.

## Key Navigation

The documentation includes links to detailed guides for both QR code types, preceded by sections on alternative payment methods like QuickPay and followed by related tokenization payment options.
