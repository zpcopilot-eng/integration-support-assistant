# Test Wallets - Zalopay Documentation

Nguồn: https://docs.zalopay.vn/docs/developer-tools/test-instructions/test-wallets

## Overview
This page provides Zalopay pre-built binaries for iOS and Android mobile platforms.

## Installation

Two versions are available for download:

- **iOS 11.7.0** (Current) - Features latest updates
- **Android 11.6.0** (Current) - Features latest updates

### iOS Setup Instructions
After successful installation, users must establish trust for the Zalopay application through device settings under "Device Management." Apple's support documentation on "[Install custom enterprise apps on iOS](https://support.apple.com/en-us/HT204460)" provides additional guidance.

### Android Setup
The Android APK can be downloaded directly from the provided link.

## Registration Process

**Step 1:** Launch the Zalopay application and authenticate using either a Zalo account or mobile number (note: one phone number links to a single sandbox account).

**Step 2:** Input the phone number and proceed.

**Step 3:** Enter verification code `111111` and complete password configuration.

## Deposit
Users must add funds to their sandbox account before conducting test transactions with Zalopay.
