# Tài liệu kỹ thuật tích hợp

Thư mục này chứa tài liệu kỹ thuật mà **Integration Support Assistant** sử dụng để trả lời câu hỏi về tích hợp hệ thống. Mọi file `.md` trong thư mục này (và các thư mục con) sẽ được index để tìm kiếm/trả lời (xem `memorySearch.extraPaths` trong `openclaw.yaml`).

## Cấu trúc đề xuất

```
zalopay-integration-docs/
  apis/                  # Tài liệu API (REST/gRPC): endpoint, request/response, mã lỗi
  integration-guides/    # Hướng dẫn tích hợp theo từng hệ thống/đối tác
  faq/                   # Câu hỏi thường gặp khi tích hợp
```

## Lưu ý
- Viết tài liệu rõ ràng, có ví dụ request/response cụ thể — assistant sẽ trích dẫn trực tiếp các phần này khi trả lời.
- Nội dung sẽ được bổ sung sau.
