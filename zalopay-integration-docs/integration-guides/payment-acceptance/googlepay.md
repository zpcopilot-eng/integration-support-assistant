# Google Pay™ Integration Guide

Nguồn: https://docs.zalopay.vn/docs/guides/payment-acceptance/googlepay/intro

## Overview

ZaloPay facilitates payments via Google Pay™ through their Gateway, which enhances customer payment options and supports secure transactions. The documentation notes that "Google Pay does not work on MacOS and iOS operating systems."

Merchants integrating Google Pay must comply with Google's policies, including their "Acceptable Use Policy" and "API Terms of Service."

## Key Features

- **Supported Networks**: MasterCard and Visa card payments
- **Payment Method**: DPAN (Tokenized card) transactions; FPAN support coming soon
- **Security**: Biometric verification available for DPAN payments, depending on the processing bank
- **Platform**: Only available through ZaloPay's hosted checkout page

## Payment Flow

The integration process involves:

1. Customer initiates checkout on ZaloPay's hosted payment page
2. Selection of Google Pay as payment method
3. Transaction completion with optional biometric authentication
4. Payment processing through ZaloPay Gateway
