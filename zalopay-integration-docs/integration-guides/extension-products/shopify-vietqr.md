# VietQR Integration Guide for Shopify

Nguồn: https://docs.zalopay.vn/docs/guides/extension-products/shopify/vietqr

## Overview

VietQR enables Shopify store owners to accept payments through QR code scanning via banking applications or Zalopay e-wallet. The integration offers key benefits including diverse payment sources, instant transaction notifications, simplified reconciliation, and quick refund capabilities.

## Integration Steps

### Step 1: Partner Registration

Merchants must register as Zalopay partners by visiting the registration portal. Existing partners should contact support at 1900 54 54 36 (branch 3) or op@zalopay.vn for integration details.

### Step 2: App Installation

The process involves:
1. Accessing the Shopify app marketplace installation link
2. Clicking to add the app and authenticating with Shopify credentials
3. Installing the application
4. Entering merchant credentials obtained from the Zalopay Merchant Dashboard, specifically the appID and Key information

Users without prior registration can complete partner setup during this step.

### Step 3: Activation

After successful credential entry, merchants are redirected to the Payment settings page where they activate VietQR. Once activated, the payment method appears in the store's payment options, completing the installation process.

## Key Features

As stated in the documentation, merchants gain "diverse payment sources" through a single QR code and receive "automatic transaction notifications" with detailed status information for cash flow management.
