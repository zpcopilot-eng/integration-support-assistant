# Online Sales Channel (OSC) - Zalopay Documentation

Nguồn: https://docs.zalopay.vn/docs/guides/extension-products/osc/intro

## Overview
OSC represents a Zalopay Commerce solution leveraging Zalo's social platform to enable merchants and customers to experience modern online shopping. The platform integrates with Zalo's OA ecosystem, offering over 10 access points to maximize business efficiency.

## Key Features

The platform provides:

1. **Store Interface** - Synchronized, attractive storefront design
2. **Order Management** - Successful orders automatically route to business systems
3. **Status Updates** - Real-time order communication between merchant and customer
4. **Payment Options** - Multiple methods including Zalopay wallet, ATM cards, Visa/Mastercard, VietQR, and ApplePay
5. **Promotion Tools** - Direct discounts, vouchers, flash sales, code entry, bundle deals, and minimum spend incentives
6. **Logistics Integration** - Connections with 3PL providers like Ahamove and Grab Express

## Integration Process

The implementation follows four sequential steps:

- **Store Configuration** - Establish storefront and configure credentials (app_id, store_id, key1, key2)
- **Menu Synchronization** - Align menus and product folders
- **Order Activation** - Enable order syncing with partner systems
- **Order Updates** - Implement real-time status tracking
