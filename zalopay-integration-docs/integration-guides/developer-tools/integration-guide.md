# Zalopay Integration Process

Nguồn: https://docs.zalopay.vn/docs/developer-tools/integration-guide

## Overview

Zalopay provides a structured integration pathway with two main environments: sandbox for testing and production for live transactions.

## Sandbox Integration

The sandbox environment serves as the initial testing phase. Merchants should:

1. **Account Setup**: "Please provide the phone number and email to the Zalopay BD to create a sandbox account"
2. **Credentials**: Zalopay supplies sandbox AppId and keys
3. **Implementation**: Review documentation and develop the integration
4. **Quality Assurance**: Submit the application to Zalopay's QC team for validation

## Production Integration

Once sandbox testing is complete, merchants move to the live environment:

1. **Account Registration**: Create a real account at the merchant registration portal
2. **Approval**: Zalopay approves the application and issues production AppId and keys
3. **Transaction Testing**: Zalopay verifies sample transactions in production
4. **Launch**: After signing off on test records, the integration goes live
